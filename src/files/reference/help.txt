============================ HELP ======================================

CLEAR       ...     CLEARS THE SCREEN

CLIENT      ...     STARTS A CLIENT

LOGOUT      ...     CLOSES CURRENT NETWORK

MAN         ...     SHOW MORE INFORMATION ABOUT A COMMAND

NOTES       ...     SENDS NOTES OVER NETWORK

SERVER      ...     STARTS A SERVER

STATUS      ...     SHOWS STATUS OF NETWORK


=================== DEVELOPER COMMANDS =================================

ADAPTER_BLOCK                  ...    BLOCK/UNBLOCK PACKETS

ADD_UPNP_MAP_AT_PORT           ...    ADDS A MAP TO MACHINE ADDRESS AT PORT

CLIENT_CONNECTION_MAXIMUM      ...    CHANGE MAXIMUM CLIENT CONNECTION

DISPLAY_NOTIFICATION           ...    DISPLAY A NOTIFICATION

DISPLAY_POPUP                  ...    DISPLAY A POPUP

FILE_BUG_REPORT                ...    FILES A BUG REPORT

LOCAL_HOST_MAXIMUM             ...    CHANGE LOCAL HOST CONNECTION MAXIMUM

MUTE_SET                       ...    CHANGE IF SOUNDS ARE MUTED

PLAY_REMIX                     ...    JUST A LITTLE FUN

REMOVE_UPNP_MAP_AT_PORT        ...    REMOVES THE PORTMAP AT A PORT

SAME_CLIENT_MAXIMUM            ...    CHANGE SAME CLIENT CONNECTION MAXIMUM

UPNP_OVERWRITE_SET             ...    CHANGE IF THE GATEWAY WILL OVERWRITE AN EXISTING MAP
